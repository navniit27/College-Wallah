## ✨ Features
* **Fully Responsive Design:** Optimized for Mobile, Tablet, and Desktop screens.
* **Modern UI:** Uses smooth wave dividers, custom shadows, and clean grid layouts.
* **Pure CSS Layouts:** Crafted using CSS Flexbox and Grid without relying on heavy frameworks.
* **Smooth Navigation:** Includes fixed header and smooth anchor link scrolling.

## 🛠️ Tech Stack Used
* **HTML5:** Semantic elements (`<header>`, `<section>`, `<footer>`).
* **CSS3:** Custom properties (variables), Media Queries, CSS Transitions.
* **Icons:** Font Awesome 4.7.
* **Fonts:** Google Fonts (Inter).
